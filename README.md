# Jumbo Birthday Surprise

A fully interactive 3D web experience designed to celebrate special moments. This application takes the user through a guided storytelling journey features Jumbo the Elephant, complete with interactive 3D elements, animations, and sound.

## Features

- **Interactive 3D Gift Box**: A physics-enabled gift box that can be opened with a tap. Automatically detects hardware capabilities and provides a 2D alternative if 3D is not supported.
- **Narrative Journey**: Sequential storytelling scenes where Jumbo presents flowers and brings the cake.
- **3D Cake Cutting**: A virtual birthday cake scene with interactive cutting mechanics and confetti celebration effects.
- **Audio Experience**: Background music helps set the celebratory mood.
- **Responsive Design**: Layouts automatically adapt to fit phones, tablets, and desktop screens.

## Technologies Used

- **React**: Core framework for user interface
- **Three.js / React Three Fiber**: For rendering the high-quality 3D scenes
- **GSAP (GreenSock)**: Used for complex timeline animations and smooth transitions
- **Vite**: Extremely fast build tool and development server
- **TypeScript**: Ensures code reliability and maintainability

## Getting Started

Follow these steps to run the project locally:

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Start Development Server**
    ```bash
    npm run dev
    ```

3.  **Build for Production**
    ```bash
    npm run build
    ```
